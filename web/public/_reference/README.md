# Ditto 프론트엔드

가치관·라이프스타일 기록 기반 매칭 SNS **Ditto**의 웹 프론트엔드입니다.

## 파일 구조

| 파일 | 설명 |
|------|------|
| `ditto_landing_v6.html` | 랜딩 페이지 (가로 스크롤, Three.js 파티클, 서비스 소개) |
| `app.html` | 앱 메인 화면 (피드, 매칭, 채팅, 프로필) |
| `login.html` | 로그인/회원가입 페이지 |

## 실행 방법

브라우저에서 HTML 파일을 직접 열면 됩니다.

```bash
# 랜딩 페이지
open web/ditto_landing_v6.html

# 앱 화면
open web/app.html
```

## 디자인 컨셉

- **가로 스크롤** 기반 랜딩 (OTB 스타일)
- **그레인 텍스처** 오버레이
- **보라색 원형** 확장 애니메이션
- **Fraunces + Pretendard** 폰트 조합
- 모바일 반응형 + 접근성 대응
